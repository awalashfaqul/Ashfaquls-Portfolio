## ğŸš€ Getting Started

1. **Clone the repository**

   ```bash
   git clone 
   cd portfolio
   ````

2. **Install dependencies**

   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run the dev server**

   ```bash
   npm run dev
   # Vite will show a local URL (e.g. http://localhost:5173)
   ```

4. **Build for production**

   ```bash
   npm run build
   npm run preview
   ```
